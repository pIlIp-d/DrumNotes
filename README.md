## Artists and Genres
multible artists or genre are seperated by '|' (Pipe)

example Line `Artist_Name-Title(DrumNotes).pdf,Title,Artist1,Metal| Rock,Drumset`

**Important** - No Spaces before the first Title, Artist, Genre,...
